### 环境配置
首先要有python3的环境
然后进入项目根目录，运行pip install -r requirements.txt

### 运行

## Linux系统下直接进入项目根目录，运行run.sh脚本即可

## Windows系统下进入weatherBriefingSpider目录，运行python start.py即可
